# spring-boot-docker

Really simple Spring Boot RESTful web service with a Dockerfile.

https://hub.docker.com/r/domjackson1/spring-boot-docker

## Run Locally
### Build
```sh
$ ./gradlew build
```

## Run application
```sh
$ ./gradlew bootRun
```

## Run tests
```sh
$ ./gradlew test
```


## Build Docker image
```sh
$ ./gradlew build docker
```
